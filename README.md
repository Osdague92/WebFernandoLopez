# 🌿WebFernandoLopez

Sitio web oficial de Luis Fernando López Pérez.

Landing page profesional desarrollada con arquitectura moderna, validación multinivel y despliegue automatizado.

---

## Descripción

Proyecto web estático optimizado para:

- Presentación profesional del autor  
- Formulario de contacto seguro  
- Formulario de contacto con Netlify Forms  
- Despliegue continuo en producción  

---

## Tecnologías

- HTML5  
- CSS3  
- JavaScript (Vanilla)  
- Bootstrap 5  
- Netlify Forms
